# Conversor de Moedas - API Node.js

API que converte valores entre moedas usando cotação atual da FreeCurrencyApi.  
Possui autenticação via token e cache de 2 minutos.

## Como rodar

```bash
git clone https://github.com/diogoferreira519/conversor-moeda.git
cd conversor-moeda

# Instalar dependências
npm install
npm run dev

#criar arquivo .env copiando o .env.example da raiz
